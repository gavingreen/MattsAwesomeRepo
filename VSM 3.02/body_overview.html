This section of the EPM2 On-line Documentation contains those background documents that it is assumed have been read prior to working with the EPM2 tools.

The EMC Solution Methods (EPM2) Overview describes the framework that helps EMC customers,  consultants, and partners develop EMC e-business solutions as quickly and successfully as possible. Based on the extensive project experience of EMC Professional Services (EPS), EPM2 uses flexible, repeatable methods and tools, and is continually improved by incorporating feedback from consultants and projects.

The EMC Solutions Methodology Team Model document provides a description of an activity-based team model for EMC Solution Methods, with recommended roles and responsibilities for accomplishing a successful EMC project.

The EPM2 Glossary is designed to acquaint the user with the terms, acronyms, and abbreviations used in the EMC Solutions Methods.
